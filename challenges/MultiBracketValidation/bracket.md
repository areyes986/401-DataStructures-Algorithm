# Multi Bracket Validation

*Author: Allyson Reyes*

---

### Problem Domain
Create a method that takes a string input and return a boolean if the input has balanced brackets.

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| `{}[]()` | TRUE |
| `}[()` | FALSE |


---
### Big O
| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
