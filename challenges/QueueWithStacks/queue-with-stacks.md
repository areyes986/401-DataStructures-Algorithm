# QueueWithStacks

*Author: Allyson Reyes*

---

### Problem Domain
Use a class named PsuedoQueue to utilize two Stack objects. Create a method that takes in a value to enqueue into the queue. 

---

### Inputs and Expected Outputs

| Input | Arg | Expected Output |
| :----------- | :----------- |
| [3]->[5]->[7] | 1 | [1]->[3]->[5]->[7] |

---
### Big O
| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
